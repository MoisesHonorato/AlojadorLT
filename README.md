# Alojador LT 🏠⚡

Sistema especializado para o gerenciamento eficiente de alojamentos destinados a equipes de trabalho em Linhas de Transmissão (LT).

---

## 🚀 Objetivo

O **Alojador LT** foi desenvolvido para centralizar e otimizar o controle de estadias, recursos e logística de alojamentos em projetos de infraestrutura elétrica. O foco do sistema é proporcionar uma interface intuitiva e ágil para gestores que precisam coordenar equipes em campo, garantindo que a logística de acomodação não seja um entrave para o progresso da obra.

## 🛠️ Tecnologias Utilizadas

O projeto utiliza o que há de mais moderno no ecossistema de desenvolvimento web para garantir performance, segurança e uma excelente experiência de uso:

-   **React 19**: Biblioteca core para construção da interface de usuário.
-   **TypeScript**: Tipagem estática para maior robustez e manutenibilidade do código.
-   **Vite**: Ferramenta de build rápida e moderna para desenvolvimento front-end.
-   **Tailwind CSS 4.0**: Framework utility-first para estilização rápida e consistente.
-   **Motion (Framer Motion)**: Para micro-interações e animações fluidas.
-   **Lucide React**: Conjunto de ícones vetoriais modernos e leves.
-   **MySQL (Backend)**: Banco de dados para persistência das informações.

## 🎨 Design System

O design do **Alojador LT** foi pensado para ser **premium, vibrante e focado na usabilidade**.

-   **Tipografia**: Utiliza a fonte **Mulish**, conhecida por sua excelente legibilidade tanto em títulos (pesos 900) quanto em textos corridos.
-   **Paleta de Cores**:
    -   `Primary Dark (#35126E)`: Transmite estabilidade e seriedade.
    -   `Primary Vibrant (#A12AC0)`: Traz modernidade e destaca elementos de ação.
    -   `Accent (#FF305C)`: Utilizada para alertas e destaques pontuais.
    -   `Button (#4D1AA1)`: Cor de destaque para chamadas à ação (CTAs).
-   **Estética**: Uso de *Glassmorphism*, sombras suaves, cantos arredondados (2xl) e transições suaves que conferem um aspecto de "aplicativo nativo" no navegador.
